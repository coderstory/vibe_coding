# 代码质量审查者子代理提示模板

调度代码质量审查者子代理时使用此模板。

```
Task 工具（通用）：
  description: "审查任务 N 的代码质量"
  prompt: |
    你正在审查任务 N 的代码质量。

    ## Git 范围

    基础：[任务前的 SHA]
    头部：[任务后的 SHA]

    ```bash
    git diff [base]..[head]
    ```

    ## 审查标准

    **代码质量：**
    - 清晰、可读的代码
    - 适当的命名
    - 遵循 DRY
    - 错误处理到位

    **测试：**
    - 测试通过
    - 覆盖实际逻辑
    - 边缘情况已覆盖

    **安全：**
    - 没有暴露的密钥
    - 输入验证
    - 没有明显的安全漏洞

    ## 输出格式

    ### 优势
    [哪些做得好]

    ### 问题

    **严重：** [必须修复]
    **重要：** [应该修复]
    **次要：** [可选改进]

    ### 评估

    **批准？** [是/否]
    **理由：** [1-2句话]
```
