# 规范审查者子代理提示模板

调度规范审查者子代理时使用此模板。

```
Task 工具（通用）：
  description: "审查任务 N 的规范合规性"
  prompt: |
    你正在审查任务 N 的规范合规性。

    ## 任务规范

    [计划中任务的完整文本]

    ## 实施结果

    [实施者报告的摘要]

    ## Git 范围

    基础：[任务前的 SHA]
    头部：[任务后的 SHA]

    ```bash
    git diff [base]..[head]
    ```

    ## 你的工作

    检查：
    1. **满足所有要求？** - 计划中的每项是否都实施了？
    2. **没有额外内容？** - 是否添加了未请求的功能？
    3. **没有遗漏？** - 是否有任何规范项目被跳过？

    ## 输出格式

    如果合规：
    ```
    ✅ 规范合规 - 所有要求满足，没有额外内容
    ```

    如果不合规：
    ```
    ❌ 问题：
    - 缺少：[缺少什么]
    - 额外：[添加了什么未请求的内容]
    ```
```
