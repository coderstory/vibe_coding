  <name>task 1: 创建知识库数据库迁移脚本</name>
  <read_first>
    - springboot/src/main/resources/db/migration/V1__init.sql
  </read_first>
  <action>
创建 V2__knowledge_base.sql 迁移脚本，包含以下表：

1. **knowledge_category** - 知识分类表
   - id (BIGINT, PK, AUTO_INCREMENT)
   - parent_id (BIGINT, DEFAULT 0) - 父分类ID，支持树形
   - name (VARCHAR(100), NOT NULL) - 分类名称
   - sort_order (INT, DEFAULT 0) - 排序
   - deleted (TINYINT, DEFAULT 0) - 软删除
   - create_time (DATETIME)
   - update_time (DATETIME)

2. **knowledge_article** - 知识文章表
   - id (BIGINT, PK, AUTO_INCREMENT)
   - category_id (BIGINT, NOT NULL) - 分类ID，外键
   - title (VARCHAR(255), NOT NULL) - 标题
   - content (LONGTEXT) - Markdown 内容
   - deleted (TINYINT, DEFAULT 0)
   - create_time (DATETIME)
   - update_time (DATETIME)
   - INDEX idx_category_id (category_id)
   - FULLTEXT INDEX ft_title_content (title, content) - 全文搜索

3. **knowledge_tag** - 标签表
   - id (BIGINT, PK, AUTO_INCREMENT)
   - name (VARCHAR(50), NOT NULL, UNIQUE) - 标签名
   - create_time (DATETIME)

4. **knowledge_article_tag** - 文章标签关联表（多对多）
   - id (BIGINT, PK, AUTO_INCREMENT)
   - article_id (BIGINT, NOT NULL)
   - tag_id (BIGINT, NOT NULL)
   - UNIQUE KEY uk_article_tag (article_id, tag_id)

5. **knowledge_file** - 知识文件表
   - id (BIGINT, PK, AUTO_INCREMENT)
   - article_id (BIGINT, NOT NULL) - 所属文章ID
   - file_name (VARCHAR(255), NOT NULL) - 原始文件名
   - file_size (BIGINT) - 文件大小（压缩前）
   - file_path (VARCHAR(500)) - 文件路径（数据库存储时为空，存储到文件时用）
   - file_data LONGBLOB - zstd 压缩后的文件数据
   - compressed_size BIGINT - 压缩后大小
   - mime_type (VARCHAR(100)) - MIME类型
   - deleted (TINYINT, DEFAULT 0)
   - create_time (DATETIME)

参考 V1__init.sql 的表结构风格，使用相同的注释格式和编码设置。

创建以下实体类，遵循 User.java 的 Lombok + MyBatis-Plus 模式：

1. **KnowledgeCategory.java**
   - 字段：id, parentId, name, sortOrder, deleted, createTime, updateTime
   - @TableName("knowledge_category")

2. **KnowledgeArticle.java**
   - 字段：id, categoryId, title, content, deleted, createTime, updateTime
   - @TableName("knowledge_article")
   - 不需要 content 的懒加载特殊处理

3. **KnowledgeTag.java**
   - 字段：id, name, createTime
   - @TableName("knowledge_tag")

4. **KnowledgeArticleTag.java** - 关联表实体
   - 字段：id, articleId, tagId
   - @TableName("knowledge_article_tag")

5. **KnowledgeFile.java**
   - 字段：id, articleId, fileName, fileSize, filePath, fileData, compressedSize, mimeType, deleted, createTime
   - @TableName("knowledge_file")

所有实体使用 @Data + @TableName + @TableId(type = IdType.AUTO) 注解。
