## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding, 表单字段间距 |
| sm | 8px | Compact element spacing, 按钮内边距 |
| md | 16px | Default element spacing, 卡片内边距 |
| lg | 24px | Section padding, 页面区块间距 |
| xl | 32px | Layout gaps, 登录页表单区域 |
| 2xl | 48px | Major section breaks, 登录页装饰区域 |
| 3xl | 64px | Page-level spacing, 布局边距 |

Exceptions: none

## Typography

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Body | 14px | 400 (regular) | 1.5 |
| Label | 14px | 500 (medium) | 1.5 |
| Heading | 18px | 600 (semibold) | 1.3 |
| Display (登录页标题) | 28px | 700 (bold) | 1.2 |

**来源**: 
- Element Plus 默认字号 14px
- base.css 现有字号 15px → 调整为 14px 符合企业后台规范
- 登录页标题使用 Display 级别突出品牌感

│     │  │  用户名输入框              │  │     │
│     │  └───────────────────────────┘  │     │
│     │  ┌───────────────────────────┐  │     │

│     │  ┌───────────────────────────┐  │     │

**设计要点:**
- 卡片居中显示 (flex 居中)
- 卡片宽度 400px，最大宽度响应式
- 卡片圆角: 8px
- 卡片阴影: 0 4px 12px rgba(0,0,0,0.1)
- 输入框之间间距: md (16px)
- 按钮高度: 40px
- 背景可添加浅色企业图案或渐变

## Interaction Contracts

### 登录页交互

| 场景 | 行为 |
|------|------|
| 输入框 focus | 边框变蓝色 #409EFF，带轻微阴影 |
| 登录按钮点击 | 显示 loading，禁用按钮 |
| 登录成功 | 跳转至 /dashboard 或首页 |
| 登录失败 | 显示错误提示 el-alert，3秒后自动消失 |
| Enter 键提交 | 触发登录提交 |

### 管理后台交互

| 场景 | 行为 |
|------|------|
| 菜单点击 | 打开对应页面，添加页签，高亮菜单项 |
| 页签点击 | 切换对应页面，高亮页签 |
| 页签关闭 | 关闭页签，切换到邻近页签 |
| 用户名点击 | 显示下拉菜单 |
| 退出登录点击 | 显示确认对话框，确认后清除 token 并跳转登录页 |
| 刷新页面 | 保持当前页签和菜单状态 |

## Responsive Behavior

| 断点 | 布局变化 |
|------|----------|
| >= 1200px | 侧边栏完全展开 200px |
| 768px - 1199px | 侧边栏可折叠，切换按钮显示 |
| < 768px | 侧边栏隐藏，汉堡菜单触发，全屏展示 |
