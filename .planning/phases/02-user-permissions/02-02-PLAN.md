  <name>task 3: 创建AOP审计日志切面 AuditAspect</name>
  <files>springboot/src/main/java/cn/coderstory/springboot/aspect/AuditAspect.java</files>
  <action>
    创建AOP切面实现AUDIT-02：系统自动记录关键数据操作（新增/编辑/删除）

    参考02-RESEARCH.md中的代码示例创建：

    ```java
    @Aspect
    @Component
    @RequiredArgsConstructor
    @Slf4j
    public class AuditAspect {
        
        private final AuditService auditService;
        
        // 拦截所有Service层的public方法（排除审计相关方法避免循环）
        @Around("execution(* cn.coderstory.springboot.service..*(..)) && !execution(* cn.coderstory.springboot.service.AuditService.*(..))")
        public Object auditAround(ProceedingJoinPoint joinPoint) throws Throwable {
            // 实现逻辑：
            // 1. 获取方法名和类名
            // 2. 从HttpServletRequest获取当前用户信息
            // 3. 从SecurityContext获取当前登录用户
            // 4. 推断操作类型（根据方法名：save/create/add->新增, update/edit->编辑, delete/remove->删除）
            // 5. 执行原方法
            // 6. 记录审计日志（userId, username, operation, target, ip）
            // 7. 返回结果
            
            // 注意：登录登出操作已经在AuthService中单独记录，这里要排除
            // 只记录数据变更操作
        }
    }
    ```

    关键点：
    - 使用@Aspect + @Component注解
    - 使用@Around增强，拦截service层方法
    - 排除AuditService自身方法避免循环调用
    - 排除AuthService的login/logout方法（已在AuthService中单独记录）
    - 从SecurityContext获取当前用户
    - 获取客户端IP地址
  </action>
