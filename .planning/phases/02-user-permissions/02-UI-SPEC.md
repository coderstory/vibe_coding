## Interaction Contracts

### User List Page

| Scenario | Behavior |
|----------|----------|
| Page load | Fetch user list with default pagination (page 1, size 20) |
| Search click | Validate form, then fetch filtered list |
| Reset click | Clear all search fields, reset to page 1 |
| Create button click | Open user create dialog |
| Edit button click | Open user edit dialog, pre-fill form with row data |
| Delete button click | Show confirmation dialog, soft delete on confirm |
| Reset password click | Open password reset dialog |
| Pagination change | Fetch corresponding page |
| Table row hover | Background highlight #F5F7FA |

### User Form Dialog

| Scenario | Behavior |
|----------|----------|
| Open (create mode) | Clear form, enable username/password fields |
| Open (edit mode) | Fill form with user data, disable username/password, hide password fields |
| Save click | Validate form, submit, close dialog on success |
| Cancel/Close click | Confirm if form dirty, then close |
| Password field | Show only in create mode |

### Role List Page

| Scenario | Behavior |
|----------|----------|
| Page load | Fetch role list |
| Search/Reset | Standard filter behavior |
| Create button | Open role create dialog |
| Edit button | Open edit dialog, pre-fill |
| Delete button | Show confirmation, soft delete |
| Assign permissions | Open permission tree dialog |

### Permission Tree Dialog

| Scenario | Behavior |
|----------|----------|
| Open | Fetch current role permissions, check corresponding nodes |
| Node check/uncheck | Standard tree behavior |
| Expand/Collapse all | Batch tree operations |
| Save click | Submit selected node IDs, close on success |

### Audit Log Page

| Scenario | Behavior |
|----------|----------|
| Page load | Fetch audit log with date range default (last 7 days) |
| Date range change | Update query |
| Operator input | Filter by operator name (fuzzy match) |
| Operation type select | Filter by type |
| Query click | Fetch filtered results |
| Reset click | Clear all filters, reset to default range |
| Empty result | Show empty state illustration |
