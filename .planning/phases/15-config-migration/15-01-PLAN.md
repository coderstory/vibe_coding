<task type="auto">
  <name>Task 1: 替换硬编码数据库用户名</name>
  <files>springboot/src/main/resources/application.yaml</files>
  <action>
将 application.yaml 中的硬编码用户名替换为环境变量：

修改 springboot/src/main/resources/application.yaml:

找到:
```yaml
datasource:
  url: jdbc:mysql://localhost:3306/admin_system?useUnicode=true&characterEncoding=utf-8&serverTimezone=Asia/Shanghai&allowPublicKeyRetrieval=true&useSSL=false
  username: root
  password: 123456
```

修改为:
```yaml
datasource:
  url: jdbc:mysql://${DB_HOST:-localhost}:3306/admin_system?useUnicode=true&characterEncoding=utf-8&serverTimezone=Asia/Shanghai&allowPublicKeyRetrieval=true&useSSL=false
  username: ${DB_USER:-root}
  password: ${DB_PASSWORD:-}
```

说明:
- `${DB_HOST:-localhost}`: 如果环境变量 DB_HOST 未设置，默认使用 localhost
- `${DB_USER:-root}`: 如果环境变量 DB_USER 未设置，默认使用 root
- `${DB_PASSWORD:-}`: 如果环境变量 DB_PASSWORD 未设置，默认使用空密码
</action>
  <read_first>
springboot/src/main/resources/application.yaml (第 45-55 行)
</read_first>
  <acceptance_criteria>
1. grep -c "username: \${DB_USER" springboot/src/main/resources/application.yaml 返回 1
2. grep -c "password: \${DB_PASSWORD" springboot/src/main/resources/application.yaml 返回 1
3. grep -c "jdbc:mysql://\${DB_HOST:-localhost}" springboot/src/main/resources/application.yaml 返回 1
4. grep -v "root\|123456" springboot/src/main/resources/application.yaml | grep -c "datasource:" 返回 0 或确认无硬编码
</acceptance_criteria>
