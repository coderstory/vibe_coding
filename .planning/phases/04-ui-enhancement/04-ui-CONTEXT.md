# Phase 4: UI 增强与主题系统

## 目标
使用 Vuestic UI 升级现有管理后台，实现动态主题、液态玻璃效果和现代交互体验。

## 需求来源
用户反馈：当前 UI 皮肤太普通，是标准的管理后台皮肤。希望引入动态效果、液态玻璃效果。

## 技术方案
- **UI 框架**: Vuestic UI (Vue 3)
- **主题切换**: 用户级别（每个用户可选择，存后端）
- **Liquid Glass**: 自定义 CSS 实现

## 技术方向
1. Vuestic UI 组件集成
2. 用户主题偏好存储（后端 API）
3. 液态玻璃 CSS 效果
4. 动态背景动画

## 已确认
- [x] 选择 Vuestic UI 作为主框架
- [x] 用户级别主题切换（存后端）
